# [sklearn机器学习实战](https://github.com/datawhalechina/machine-learning-toy-code)

本项目通过sklearn学习库实现一些常见的机器学习算法，来帮助读者分析各个算法的特点。

|          | 支持向量机 |      |      |
|          |   决策树   |      |      |
